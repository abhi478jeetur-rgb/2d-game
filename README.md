# 2D Game

A 2D game project built with HTML5, CSS3, and JavaScript. This is an interactive browser-based game with smooth animations and responsive gameplay mechanics.

## Features

- **HTML5 Canvas** - Smooth 2D graphics rendering
- **CSS3 Styling** - Modern and responsive UI design
- **JavaScript Game Logic** - Event-driven player controls and game mechanics
- **Collision Detection** - Advanced physics for game interaction
- **Score System** - Track player progress and achievements
- **Responsive Design** - Works seamlessly on desktop and mobile devices

## Technologies Used

- **Frontend**: HTML5, CSS3, Vanilla JavaScript
- **Rendering**: HTML5 Canvas API
- **Styling**: CSS3 Flexbox and Grid

## Getting Started

### Prerequisites

- Modern web browser (Chrome, Firefox, Safari, Edge)
- Basic knowledge of web development (optional)

### Installation

1. Clone the repository
```bash
git clone https://github.com/abhi478jeetur-rgb/2d-game.git
cd 2d-game
```

2. Open the game in your browser
```bash
open index.html
# or
start index.html
```

## How to Play

- Use **Arrow Keys** or **WASD** to move your character
- **Spacebar** to jump
- Avoid obstacles and collect items to increase your score
- Try to reach the highest level possible

## Game Mechanics

### Player Movement
- Smooth keyboard controls with acceleration and deceleration
- Gravity simulation for realistic jumping physics

### Obstacles
- Dynamic obstacles that increase in difficulty
- Collision detection prevents player from passing through walls

### Scoring System
- Points for collecting items
- Bonus multipliers for consecutive achievements
- Leaderboard to track best scores
